what different types of JavaScript Events can allow you to do.

I will be using the button as my element and will be changing the event using the JavaScript Function.

Using onClick allows you to push the button and the date will appear. button onClick="displayDate()".

Using onMouseover the date will appear when you run your mouse over the button. button onMouseover="overDate()".

Using onMouseout the date will appear when you take the mouse off of the button. button onMouseout="offDate()".

Using onKeydown you click the button and then push your down arrow.

This JavaScript event runs directly from the HTML body using onclick.
