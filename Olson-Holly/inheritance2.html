JavaScript Inheritance

When you use inheritance you are using a prototype of what you have done. You have your code written out with your properties and values you want to use. If you want to add a property you simply take the function name that has the parameters, properties, and put a . then prototype then . then the new property you want to add. Person.prototype.attitude = "Amazing";. Then when you call the document you can make it play out the new property value like this document.getElementById.innerHTML= "My sister is" + mySister.attitude; and it will show it on the screen when you push the button.
