API storage

You can use local storage to let your computer browser store data for you.

You can use session storage to let your computer browser store data for you. Thought this only lasts for the session, while the tab is open.

Using local storage click the button to see the counter increase.

because this is a a Local Storage when you close the tab and re open it, the count will start where you left the number of clicks off the last time.

Using session storage click the button to see the counter increase.

Using session storage you close the browser and open it again and the counting will start again from 1 as you are starting a new session.

When you push the button you will see the local storage members names.

When you push the button you will be asked your favorite animal. When you put it in you will then see all the Local stored animal names plus the one you added.
